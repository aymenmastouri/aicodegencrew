# Issue Summary: VVZ-2998

**Impact Level:** HIGH
**Type:** Bug
**Estimated Timeline:** short

## Summary

After importing a JSON file, the system loses some of the involved parties (e.g., the organisation "Sparkasse") when you try to edit the record. This makes the entry appear incomplete and prevents you from saving the record.

## Workaround

Re‑import the JSON file each time before editing, or manually add the missing parties again in the edit view.
