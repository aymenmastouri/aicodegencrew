# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

UVZ currently runs on Angular 18 with Pattern Library (PL) 11.3.1. Angular 18 reached end of security support on 21.11.2025, meaning without costly 'Never-Ending Support', the application will no longer receive critical security patches — exposing users to unpatched vulnerabilities. To maintain security, compliance, and access to modern features and tooling, the system must be upgraded to Angular 19 and PL 12.6.0. This is not optional: delaying it risks exposure to security threats, non-compliance, and increasing technical debt that will make future upgrades harder or impossible.
