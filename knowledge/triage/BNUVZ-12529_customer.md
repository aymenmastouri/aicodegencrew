# Issue Summary: BNUVZ-12529

**Impact Level:** MEDIUM
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ front‑end currently runs on Angular 18 with Pattern Library 11.3.1. Security support for Angular 18 ends on 21‑Nov‑2025, after which no free patches are provided. To keep the application safe, compliant and maintainable, the front‑end must be upgraded to Angular 19 and Pattern Library 12.6.0, together with all required runtime dependencies. If the upgrade is not performed, the system will become vulnerable to unpatched security issues and the team would have to pay for a costly Never‑Ending Support contract to receive any fixes.

## Workaround

Continue using the paid Never‑Ending Support for Angular 18, but this incurs additional cost and does not solve the underlying obsolescence.
