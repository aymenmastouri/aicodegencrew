# Issue Summary: BNUVZ-12529

**Impact Level:** MEDIUM
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ application currently runs on Angular 18 and Pattern Library (PL) 11.3.1. Security updates for Angular 18 stop on 21‑Nov‑2025, after which only a paid "Never‑Ending Support" option is available. To keep the system secure, compliant and avoid extra licensing costs, the application must be upgraded now to Angular 19 together with PL 12.6.0, and all related dependencies must be aligned. If the upgrade is not performed, the product will become vulnerable to unpatched security issues and may breach contractual security requirements.

## Workaround

Continue using Angular 18 with paid Never‑Ending Support, but this incurs additional cost and does not solve the underlying security lifecycle issue.
