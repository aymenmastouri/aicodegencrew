# Issue Summary: BNUVZ-12529

**Impact Level:** CRITICAL
**Type:** Enhancement/Task
**Estimated Timeline:** long

## Summary

The UVZ application currently runs on Angular 18 with Pattern Library (PL) 11.3.1. Angular 18’s security support ended on 21.11.2025, leaving the system exposed to unpatched vulnerabilities unless costly 'Never-Ending Support' is purchased. To avoid technical debt, security risk, and future compatibility issues, the application must be upgraded to Angular 19 and PL 12.6.0 — a mandatory step before any future Angular upgrades (e.g., Angular 20/21). This is not optional: staying on Angular 18 violates the organization’s security and maintenance policies and will block future development and compliance efforts.
