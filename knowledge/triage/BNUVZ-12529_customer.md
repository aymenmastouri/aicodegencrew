# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ web application currently runs on Angular 18 and Pattern Library 11.3.1. Security support for Angular 18 ends on 21 Nov 2025, after which only a paid "Never‑Ending Support" option is available. To keep the system secure, compliant and avoid extra licensing costs, the application must be upgraded now to Angular 19 and Pattern Library 12.6.0. This upgrade also aligns the UI stack with the versions used by other Bnotk products and prepares the codebase for future Angular releases.

## Workaround

Continue with paid Never‑Ending Support for Angular 18, but this incurs additional cost and does not solve the underlying security risk.
