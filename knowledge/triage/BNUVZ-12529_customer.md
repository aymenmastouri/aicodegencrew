# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ application currently runs on Angular 18 and Pattern Library (PL) 11.3.1. Security support for Angular 18 ends on 21‑Nov‑2025, after which only paid "Never‑Ending Support" is available. To keep the product secure, receive free security patches, and stay on a supported stack, the application must be upgraded now to Angular 19 and PL 12.6.0. Without this upgrade the system would become vulnerable to unpatched security issues and could eventually violate compliance requirements.

## Workaround

Continue using paid Never‑Ending Support for Angular 18, but this incurs additional cost and does not solve the underlying security risk.
