# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ application is currently running on Angular 18 with Pattern Library (PL) 11.3.1, but Angular 18’s official security support ended on 21.11.2025. Without upgrading, the system loses access to critical security patches and updates unless expensive 'Never-Ending Support' is purchased. This task upgrades Angular to v19 and PL to v12.6.0 — the minimal necessary step before planning future upgrades (e.g., to Angular 20/21), ensuring continued security, compliance, and maintainability. Failure to act risks unpatched vulnerabilities, technical debt accumulation, and increased cost/complexity of future upgrades.
