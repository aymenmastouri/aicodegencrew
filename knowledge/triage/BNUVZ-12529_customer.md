# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ web application currently runs on Angular 18 and Pattern Library 11.3.1. Security updates for Angular 18 stop on 21‑Nov‑2025, and the only way to keep receiving patches is to pay for a Never‑Ending Support contract. To stay secure and compliant, the application must be upgraded now to Angular 19 and Pattern Library 12.6.0. This upgrade also brings the required updates of Node.js, TypeScript and several UI libraries. If the upgrade is not performed, the system will become vulnerable to unpatched security issues and may eventually be forced to run on an unsupported stack, risking compliance breaches and potential downtime.

## Workaround

Continue paying for Never‑Ending Support on Angular 18 (or accept the lack of security patches).
