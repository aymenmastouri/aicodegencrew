# Issue Summary: BNUVZ-12529

**Impact Level:** MEDIUM
**Type:** Enhancement/Task
**Estimated Timeline:** long

## Summary

UVZ currently runs on Angular 18 and Pattern Library (PL) 11.3.1 — but Angular 18 lost official security support on 21.11.2025. Without upgrading, the system becomes vulnerable to unpatched security issues and will rely on expensive 'Never-Ending Support' for any future fixes. This task updates Angular to version 19 and PL to 12.6.0 to restore full support, compliance, and roadmap alignment. Failure to act risks security exposure, loss of vendor support, and increased technical debt.

## Workaround

Continue using Angular 18 with paid Never-Ending Support — not sustainable long-term.
