# Issue Summary: BNUVZ-12529

**Impact Level:** MEDIUM
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ web application currently runs on Angular 18 and Pattern Library 11.3.1. Security support for Angular 18 ends on 21 Nov 2025, after which no free patches are provided. Continuing to use an unsupported framework would expose the product to unpatched vulnerabilities and would force the business to pay for a Never‑Ending Support contract. Upgrading to Angular 19 and Pattern Library 12.6.0 restores regular security updates, keeps the application compliant with internal security policies, and ensures that the UI stays compatible with the latest browser features.

## Workaround

Remain on Angular 18 by purchasing Never‑Ending Support from the vendor, but this incurs additional cost and does not eliminate the underlying security risk.
