# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ web application currently runs on Angular 18 and Pattern Library 11.3.1. Security support for Angular 18 ends on 21‑Nov‑2025, after which only paid "Never‑Ending Support" will provide patches. To keep the product secure, compliant and maintainable, the UI must be upgraded to Angular 19 and Pattern Library 12.6.0 now. If the upgrade is not performed, the application will no longer receive free security updates, increasing the risk of vulnerabilities and eventually forcing a costly support contract or a forced migration later.
