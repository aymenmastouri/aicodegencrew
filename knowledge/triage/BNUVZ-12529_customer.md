# Issue Summary: BNUVZ-12529

**Impact Level:** MEDIUM
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ application currently runs on Angular 18 and Pattern Library 11.3.1. Security support for Angular 18 ends on 21‑Nov‑2025, after which no free patches are provided. To keep the system safe and avoid paying for the costly Never‑Ending Support subscription, the frontend must be upgraded to Angular 19 and Pattern Library 12.6.0 now. The upgrade also brings the UI libraries to versions that are compatible with the new framework, ensuring continued functionality of the vertical action bar and other UI components.

## Workaround

Continue paying for the Never‑Ending Support subscription to receive security patches for Angular 18, but this incurs additional cost and is only a temporary measure.
