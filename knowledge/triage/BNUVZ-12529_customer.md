# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** long

## Summary

The UVZ application is currently built on Angular 18 and Pattern Library (PL) 11.3.1. Angular 18’s security support ended on 21.11.2025, meaning only paid 'Never-Ending Support' remains for updates — a risky and unsustainable path. To avoid future vulnerabilities, compliance gaps, and vendor lock-in, UVZ must upgrade to Angular 19 and PL 12.6.0 to re-enter the supported ecosystem. This is not optional; it is a time-sensitive operational necessity to maintain system integrity and release velocity.
