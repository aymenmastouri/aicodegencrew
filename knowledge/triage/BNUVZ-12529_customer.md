# Issue Summary: BNUVZ-12529

**Impact Level:** MEDIUM
**Type:** Enhancement/Task
**Estimated Timeline:** short

## Summary

The UVZ web application currently runs on Angular 18 and Pattern Library 11.3.1. Security updates for Angular 18 stop on 21‑Nov‑2025, after which only paid "Never‑Ending Support" is available. To keep the system secure, compliant and avoid extra licensing costs, the application must be upgraded now to Angular 19 and Pattern Library 12.6.0. This upgrade also aligns the UI stack with the roadmap and ensures continued compatibility with third‑party UI components.

## Workaround

Continue using Angular 18 with paid Never‑Ending Support, but this incurs additional cost and leaves the system on an unsupported stack after the security‑support deadline.
