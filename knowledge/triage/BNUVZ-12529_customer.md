# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** short

## Summary

The UVZ front‑end currently runs on Angular 18 and Pattern Library 11.3.1. Security support for Angular 18 ends on 21 Nov 2025, and the only way to keep receiving patches would be to pay for a never‑ending support contract. To stay secure and avoid extra licensing costs, the application must be upgraded now to Angular 19 and Pattern Library 12.6.0, which are still under standard support. If the upgrade is not performed, the system will become vulnerable to unpatched security issues after the support deadline, potentially exposing user data and violating compliance requirements.
