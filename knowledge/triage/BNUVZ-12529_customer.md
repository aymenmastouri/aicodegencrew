# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** short

## Summary

The UVZ front‑end currently runs on Angular 18 and Pattern Library 11.3.1. Security patches for Angular 18 stop on 21 Nov 2025, after which only a paid "Never‑Ending Support" option is available. To keep the application secure, compliant and maintainable, the front‑end must be upgraded now to Angular 19 together with Pattern Library 12.6.0 and all related dependencies. If the upgrade is not performed, the system will become vulnerable to unpatched security issues and may eventually require costly support contracts or a forced migration later under tighter time pressure.

## Workaround

Continue using Angular 18 by purchasing the paid Never‑Ending Support from the vendor, but this incurs additional cost and does not solve the underlying obsolescence.
