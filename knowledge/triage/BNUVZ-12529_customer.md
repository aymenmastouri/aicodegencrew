# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ application currently runs on Angular 18 and Pattern Library (PL) 11.3.1, but Angular 18 reached end-of-life for security updates on 21.11.2025. Continuing to use Angular 18 would leave the system exposed to unpatched vulnerabilities unless expensive 'Never-Ending Support' is purchased. This task upgrades Angular to version 19 and PL to 12.6.0, ensuring continued security support, compatibility with modern tooling, and future upgradeability (e.g., to Angular 20/21). Failure to upgrade risks security exposure, loss of vendor support, and technical debt accumulation.
