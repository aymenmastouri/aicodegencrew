# Issue Summary: BNUVZ-12529

**Impact Level:** HIGH
**Type:** Enhancement/Task
**Estimated Timeline:** medium

## Summary

The UVZ web application currently runs on Angular 18 and Pattern Library 11.3.1. Security support for Angular 18 ends on 21‑Nov‑2025, after which no free patches are provided. To keep the system safe, compliant and maintainable, the UI must be upgraded to Angular 19 and Pattern Library 12.6.0. If the upgrade is not performed, the application will become vulnerable to unpatched security issues and the organization would have to pay for a Never‑Ending Support contract, incurring additional cost and operational overhead.

## Workaround

Continue using Angular 18 under a paid Never‑Ending Support contract, but this incurs extra cost and does not eliminate the underlying security risk.
