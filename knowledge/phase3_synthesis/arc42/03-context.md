## 3.1 Business Context

### System Overview

The **uvz** platform is a backend‑centric solution that orchestrates deed‑entry processing, key‑management, archiving and business‑purpose services. It exposes a rich **REST API** consumed by internal front‑ends (Angular SPA, Node.js `jsApi`) and by external partner systems via HTTPS. The system stores its state in a PostgreSQL database and publishes domain events on a Kafka bus for downstream consumers.
